# Config files


## Tmux
- Copy to ~/.tmux.conf
- Install the TPM plugin manager: https://github.com/tmux-plugins/tpm
- Install Catpuccin themes plugin: https://github.com/catppuccin/tmux
- Install Nerd fonts: https://www.nerdfonts.com/font-downloads (I use 0xProto)
